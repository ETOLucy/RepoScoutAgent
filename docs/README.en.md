## Maintenance Rules

Default Markdown files without a language suffix are Simplified Chinese; English mirrors use
`.en.md`. User-visible features must update the root README, the documentation index, and the
relevant specialist document. Historical performance baselines remain intact and are explicitly
marked when current behavior changes.
